# DjangoSampleProject

Simple Imageboard engine. Allows to add threads with numerous posts.
First three posts are shown for each thread. 

### including
* Django-Bootstrap 3 powered


## Getting Started
```
virtualenv -p python3.6 venv
. venv/bin/activate
pip install -r requirements.txt
```
